## Guess My Number Game

A number guessing game where the player tries to guess a secret number.

🎮 Features
1. Score tracking
2. Highscore system
3. Hints (too high / too low)

🛠️ Technologies Used
1. HTML
2. CSS
3. JavaScript

▶️ How to Run
1. Open the project folder
2. Double click "index.html"

OR (recommended):
Use VS Code Live Server
